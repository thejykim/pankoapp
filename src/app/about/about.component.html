About the project

This is a kanban project that collects basic information while you use it, and returns statistics and recommendations based on calculated habits. Its aim is to increase productivity in a research-backed manner. The project also places an emphasis on streamlined usage with minimal clutter.

The entire code base is available on GitHub, and contributions are encouraged. This is a tool that is powered by the community. Thus, bar immense hosting fees, all features will remain free.
